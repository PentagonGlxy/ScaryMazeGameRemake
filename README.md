## üöÄ How to Run

### 1. Prerequisites
You need Python installed on your computer. You also need the `pygame` library.
pip install pygame
2. Assets Setup
Important: To make the game work on your machine, ensure you have the sound and image files properly linked in the code.
Create a folder named assets (or similar) in the same directory as the script.
Place your jumpscare image and sound file inside.
Update the file paths in the scary_maze_final.py script to match the location of your files.
3. Running the Game
Click the ScaryMaze application.
